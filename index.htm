HTML


 

HTML znamena: Hyper Text Markup Language

HTML je programovaci Jazyk pro Psani (neni to programovani) Webu

Zaklady v kodu HTML vypada Takhle:

<html>
  <head>
  <title>
  </title>
  </head>
  <body>
  </body>
  </html>
 

Samorejmne tam tento kod nemusi bejt ale lip se pak orientuje
